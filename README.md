# Spring Boot Login Application

This is a simple login application built with Spring Boot.
The purpose of this project is to demonstrate backend authentication
using Java, Spring Boot, and AWS RDS.

## Tech Stack
- Java
- Spring Boot
- HTML / CSS / JavaScript
- AWS RDS (MySQL)

## Development Environment
- Backend: IntelliJ IDEA
- Frontend & configuration: Visual Studio Code
- The frontend is currently implemented with plain HTML/CSS/JavaScript
  and is planned to be migrated to React in a future update.

## Features
- User login
- Password hashing (BCrypt)
- REST API based authentication

## Purpose
This project is created as a portfolio application.
